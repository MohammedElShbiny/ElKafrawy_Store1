# تعليمات النشر على Netlify (النسخة المجانية)

هذا المشروع مصمم ليعمل كـ Single Page Application (SPA). للنشر المجاني على Netlify، اتبع الخطوات التالية:

## 1. إعدادات البناء (Build Settings)
عند ربط مستودع GitHub بـ Netlify، استخدم الإعدادات التالية:
- **Build Command:** `npm run build`
- **Publish Directory:** `dist`

## 2. حل مشكلة الروابط (Direct Links Fix)
بما أن التطبيق يستخدم React Router، فإن الانتقال المباشر للروابط (مثل `/admin` أو `/cart`) قد يؤدي لخطأ 404. تم إضافة ملف `public/_redirects` تلقائياً لحل هذه المشكلة.

## 3. المتغيرات البيئية (Environment Variables)
يجب إضافة مفتاح API الخاص بـ Gemini لضمان عمل ميزات الذكاء الاصطناعي:
1. اذهب إلى **Site settings** > **Build & deploy** > **Environment variables**.
2. أضف متغير جديد باسم: `VITE_GEMINI_API_KEY`.
3. ضع القيمة الخاصة بمفتاحك.

## 4. قيود النسخة المجانية
- خادم Express (`server.ts`) لا يعمل مباشرة على استضافة Netlify الاستاتيكية.
- التطبيق مبرمج ليعمل بشكل مستقل باستخدام البيانات المحلية في حال عدم توفر الخادم.
- للبيانات الدائمة، يوصى بربط التطبيق بـ **Firebase** أو **Supabase**.
